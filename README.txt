###############################################
            	Title
###############################################
Image Processing GUI (aka Find my Photo)


###############################################
            	Description
###############################################
Our program uses GUI for python software that 
detects if an uploaded cropped image is in a 
global picture. 


###############################################
            	How To Run
###############################################
You need python installed in your computer.
To run the file, open your terminal (if you are
a windows user, type cmd in your search bar).
Go to the main.py file path, and type:
>>> py main.py
The program should be opened.
Upload the global image into the GUI, then
the cropped image, and press execute.
If the image is more than 500x500, then expect
to wait for more than a couple of minutes.
If successful, a message would pop out saying
that it has in fact successfully found the 
cropped image in the global image, displaying
the cropped image with red filter on the
global image. The terminal should also indicate
the coordinate of the first pixel of where
the cropped image is in the global image.
